# Chat-App

**Author:** Adnan Pathan  
**Project Type:** Real-Time Chat Application  
**Languages/Technologies Used:** HTML, CSS, JavaScript, Node.js, and Socket.IO

## Project Overview
Chat-App is a real-time chat application that enables users to communicate seamlessly. It leverages the power of WebSockets via Socket.IO to provide instant message delivery. This application is ideal for anyone looking to integrate a simple yet powerful communication tool into their web projects.

## Features
- Real-time messaging using Socket.IO
- Responsive UI built with HTML and CSS
- Secure and scalable Node.js backend
- Lightweight and easy to set up

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/iadnan172/Chat-Room.git
   cd Chat-Room
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the application:
   ```bash
   node server.js
   ```

4. Open your browser and navigate to:
   ```
   http://localhost:8000
   ```

## How to Use
1. Open the application in your browser.
2. Enter your username to join the chat room.
3. Start chatting with other users in real-time.

## Prerequisites
- Node.js (v14+)
- npm (v6+)
